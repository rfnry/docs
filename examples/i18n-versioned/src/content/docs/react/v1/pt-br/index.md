Este é um template Astro mínimo e opinativo para documentação. Substitua este conteúdo pelo seu próprio.

## O que há neste template

Um cabeçalho com espaço para logo, seletor de versão, busca, seletor de idioma e alternador de tema. Uma sidebar em árvore à esquerda, seu conteúdo no centro e um índice da página à direita.
